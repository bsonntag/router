# Link component

Renders an `<a>` element that links to a given path.

## Props

| Prop     | Type    | Required |
| -------- | ------- | -------- |
| children | Node    | No       |
| replace  | boolean | No       |
| state    | any     | No       |
| to       | string  | Yes      |

## Usage

```js
<Link to={'/users'}>Go to users</Link>
```
